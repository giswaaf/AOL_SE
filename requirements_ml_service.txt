# MIGRATION GUIDE — Cara upgrade dari versi lama
# ============================================================
#
# 1. Install dependencies baru:
#    pip install insightface>=0.7.3 onnxruntime>=1.16.0 deepface>=0.0.79 -r requirements.txt
#
# 2. Pertama kali jalankan, InsightFace akan download model buffalo_l (~300MB).
#    Pastikan koneksi internet tersedia saat startup pertama.
#    Atau pre-download manual:
#      python -c "from insightface.app import FaceAnalysis; FaceAnalysis('buffalo_l').prepare(ctx_id=0)"
#
# 3. ⚠️  WAJIB: Re-enroll semua mahasiswa!
#    Embedding lama (9216-dim pixel flatten) TIDAK kompatibel dengan
#    embedding baru (512-dim ArcFace). Jalankan script migrasi:
#
#      python scripts/migrate_clear_old_embeddings.py
#
#    Kemudian minta semua mahasiswa melakukan pendaftaran ulang via
#    fitur "Pendaftaran Video" di halaman profil.
#
# 4. Copy file-file baru ke posisi yang tepat:
#    server/ml-service/app/ml/face_encoder.py      (ganti yang lama)
#    server/ml-service/app/ml/face_matcher.py      (ganti yang lama)
#    server/ml-service/app/ml/attendance_pipeline.py (ganti yang lama)
#    server/ml-service/app/api/routes/face.py      (ganti yang lama)
#    faceattend/frontend/src/students/components/FaceLivenessEnrollment.jsx (BARU)
#
# 5. Update StudentProfile.jsx — tambahkan FaceLivenessEnrollment:
#    import FaceLivenessEnrollment from "../components/FaceLivenessEnrollment";
#
#    Ganti card "Face Image Upload" yang lama dengan:
#    <FaceLivenessEnrollment
#      studentId={data.id}
#      token={localStorage.getItem("token")}
#      onSuccess={() => queryClient.invalidateQueries(["myStudentProfile"])}
